## Hızlı Referans (CSS Variables)

```css
:root {
  /* Ana Palet */
  --tt-red:          #C90C0F;
  --tt-black:        #000000;
  --tt-white:        #FFFFFF;

  /* Sunum Paleti */
  --tt-navy:         #000C54;
  --tt-deep-navy:    #1C1D52;
  --tt-blue:         #0035D6;
  --tt-purple:       #1E122F;
  --tt-pink:         #CB29AC;
}
```
